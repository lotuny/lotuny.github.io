重要概念

  • 线程安全
    指的是一个进程管理下的多线程并行运行时的数据统一问题。举个例子，对于同一个变量a，如果存在多线程同时读取写入的情况，则可能产生不真实的数据。

    StringBuffer是线程安全的，StringBuilder则是线程不安全，但是在单线程下性能较前者好。
